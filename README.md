# WSLfantasy
Historical Comparative Analysis on Professional Surfers


Within this repository are the two data sets used.

trueSurfData.csv 
Holds surfer:
-name (ID)
-Event ID
-scores

EventDetails.csv
Holds event:
-Event ID
-event_year
-month
-start_date
-end_date
-event_name
-beach
-province
-country
